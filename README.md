# 🏥 Al-Shifa Clinic - Complete Healthcare Management System

A full-stack platform with real-time video consultations, appointment management, and comprehensive healthcare features.

## ✨ Features

### 🩺 Core Functionality
- **Multi-role Authentication** (Patient, Doctor, Admin)
- **Real-time Video Consultations** with Socket.io integration
- **Appointment Booking System** with Stripe payments
- **Electronic Prescription Management**
- **Doctor Rating & Review System**
- **Multi-language Support** (English & Arabic with RTL/LTR)

### 👥 User Roles
- **Patients**: Book appointments, video consultations, prescriptions, ratings
- **Doctors**: Manage schedule, consultations, patient records, prescriptions
- **Admin**: Dashboard analytics, user management, system oversight

### 💼 Technical Highlights
- **Full-stack TypeScript** application
- **MongoDB** with Mongoose ODM
- **React 19** with Tailwind CSS
- **Socket.io** for real-time features
- **JWT authentication** with refresh tokens
- **Cloudinary** for image storage
- **Stripe** payment integration
